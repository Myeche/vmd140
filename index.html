Course Overview

This is a hands-on, introductory-level course covering production techniques and best practices for front-end web development. Students will learn how to create well-structured, standards compliant web documents using Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS) to create user-friendly interfaces for a range of web enabled devices. Topics will include employing efficient workflow practices, available tools and resources for web developers, current production techniques, and more.
